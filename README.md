# Representacao-Soma-e-Subtracao-de-Vetores-no-Manim

Este projeto consiste no desenvolvimento de uma animação utilizando a biblioteca Manim, com o objetivo de visualizar e explicar operações vetoriais fundamentais, amplamente utilizadas em computação gráfica, desenvolvimento de jogos, simulações e animações.

[Assista ao vídeo da animacão](https://youtu.be/ycz1KbVW-wc)

A animação foi desenvolvida como atividade acadêmica e busca transformar conceitos matemáticos abstratos em representações visuais claras e intuitivas, facilitando a compreensão de temas como adição e subtração de vetores no plano cartesiano.

## Objetivos do Projeto
- Representar vetores de forma visual e didática
- Demonstrar adição e subtração de vetores passo a passo
- Relacionar conceitos matemáticos com aplicações práticas da computação gráfica
- Auxiliar no aprendizado por meio de animações interativas
- Servir como material de apoio para disciplinas como:
  - Computação Gráfica
  - Matemática Vetorial
  - Álgebra Linear
  - Desenvolvimento de Jogos

## Conceitos Abordados
- Vetores no plano cartesiano
- Vetor posição e vetor deslocamento
- Adição de vetores (v₁ + v₂)
- Subtração de vetores (v₂ − v₁)
- Representação gráfica de pontos e deslocamentos
- Interpretação geométrica das operações vetoriais

Além disso, são apresentados exemplos práticos, como:
- Combinação de movimentos (ex: movimento do jogador + vento)
- Cálculo de direção e deslocamento entre dois pontos
- Aplicações em movimentação de sprites e câmeras em jogos
